Topic 1

To get started with writing Java, open the IDE and write your first "Hello world" Java code:

System.out.print("Hello World");

Hit "run" and watch the code appear in your console!

  • UseSystem.out.println();to create a new line after your statement.
  • The code printed to the console is a data type called a String.
