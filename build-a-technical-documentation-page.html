Hello World

Java is one of the world's most popular programming languages. It was released in 1995 to serve as a programming language that would run code consistently on any machine.

Java is an object-oriented programming language. An object is an individual instance of a class, which is a section of code that defines the structure of objects. In Java, your code will not compile unless it is a part of a class. For example, the following code would not run:

System.out.print("Hello World");

In order to get this code to run, we have to put it in a class that contains a main method. The main method tells Java where to start running your code. Most of the time, your code will also be inside of a package, which defines where your code is saved on your computer, but it is not necessary for your code to compile. The following is what a complete .java file might look like for a "Hello World" program.

package helloWorld;

public class HelloWorld {

public static void main(String[] args) {
//this is the main method
System.out.print("Hello World");
}
}
Variables and Data Types

Here are 5 basic data types in Java

  • Integers
  • Floats and Doubles
  • Characters
  • Strings
  • Booleans

Integers are whole numbers. Some examples include 13, 0, -1, 7, and 903.

Floats and doubles are numbers with decimals. Doubles can store numbers that are twice as long as floats, so use doubles if you anticipate using large numbers or need a high degree of precision. Some examples include 2.71828, 1.00, 39.99, -17.3019, and 0.0.

Characters are single characters. Examples include a, z, 1, #, and å­—.

Strings contain one or more characters. Examples include abc, hello world, p@55w0rd, z, and 123.

Booleans are a special data type that are always only ever True or False

Conditionals

Conditionals are code that only run when certain conditions are met.

if (coinToss == True) {
System.out.print("Heads");
}
else {
System.out.print("Tails");
}
Loops

Loops are sections of code that run repeatedly until a condition is broken.

There are two types of loops: while loops and for loops. While loops are generally used when the number of times the loop will repeat is unpredictable, otherwise for loops are preferred.

The following code shows how to use either type of loop to calculate a factorial.
int factorial = 1;
int x = 10;
while (x > 1) {
factorial = factorial * x;
x = x - 1;
}

for (int x = 10; x > 1; x = x - 1) {
factorial = factorial * x;
}

Note that the variables are declared outside of the while loop but inside the for loop. Because of this, when using a for loop, the variables will not be accessible outside of the loop.

Putting it all Together

Now you should be familiar with the basics of programming with Java. While the syntax may vary from language to language, these basic principles should hold true across most programming languages. Good luck and happy coding!
